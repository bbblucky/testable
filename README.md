# Cucumber & Capybara Tests for teachable demo website
## Task

The demo website has a test school with a course that contains 4 lecture.
The tests are focusing following function: 
- Sign up and log in
- Enroll the course
- Complete the course

## Setup

```
gem bundler install
```

```
bundle install
```
